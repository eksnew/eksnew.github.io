Body Part-Based Representation Learning for Occluded Person Re-Identification

Posted on 2024-04-07 Edited on 2024-04-08 In 论文阅读

WACV 2023的工作。重识别方向的。

1. 引入

1.1. 现有问题

  • 个体身体部位的外观不如全局外观具有区分性（两个不同的ID可能具有相同的局部外观）；
  • ReID数据集没有提供人体拓扑注释。

1.2. what to do

提出了一种body part-based的ReID模型BPBreID，设计了两个模块，用于预测身体部位的注意力图和生成body part-based的ReID目标特征。 提出了一种新颖的训练方案GiLt，用于学习对遮挡和非判别性局部外观具有鲁棒性的基于部件的表示。

2. 模型与方法论

2.1. Body Part Attention Module 身体部位注意力模块

身体部位注意力模块接受由主干网络提取的特征图作为输入，并输出一组突出显示ReID目标身体部位的注意力图。

2.3.1. GiLt Loss

GiLt loss由身份分类损失(identity classification loss)和自定义的部分平均三元组损失(part-averaged triplet loss)组成。 与其他流行的part-based的方法不同，因为存在遮挡和不具有区分性的局部外观，所以模型不在part-based的特征上应用identity loss。 因为part-based的特征并不总是足够的辨别力来识别一个人，这使得identity prediction的目标无法实现。 所以在这样的局部表示上添加identity loss会破坏性能。（合理的）

不过类似于大多数最先进的ReID方法，模型仍然通过将identity loss supervision应用于整体特征来受益。

其次，模型通过自定义的part-averaged triplet loss在part-based的特征上应用triplet loss constraint。 在推断阶段，样本间的距离将使用part-based的特征进行计算，因此直接使用triplet loss constraint来优化它们的相对距离是有意义的。 然而因为存在遮挡，所以不应在整体embedding上强制执行triplet loss constraint。 如果两个相同identity的整体embedding中有至少一个被部分遮挡，二者将具有内在不同的表示（因为每个embedding会代表目标身体的不同子集）， 因此若使用triplet loss将这两个不同的整体特征拉近到特征空间中，将对性能产生破坏性的影响。（合理的）

总之论文认为part-based的方法的最佳训练策略是：

  • 仅对整体特征应用identity loss constraint；
  • 仅对part-based的特征应用triplet loss constraint（通过自定义的part-averaged triplet loss）。

论文将这种策略称为Global-identity Local triplet(GiLt)：

是带label smoothing和BNNeck trick的cross-entropy loss， 是part-averaged triplet loss。 优化网络以从每个整体embedding中预测输入样本的identity。
